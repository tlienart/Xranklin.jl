Setup

Version of the package used in the example:

1.5.2

To install dependencies on Github Action add a run step prior to the build with:

sudo apt-get update -qq
sudo apt install -y pdf2svg texlive-latex-base texlive-binaries texlive-pictures texlive-latex-extra texlive-luatex

Example

using PGFPlotsX

x = range(0, pi, length=500)
y = @. sin(exp(x)) * sinc(x)

@pgf PGFPlotsX.Axis(
    Plot(
        {no_marks},
        Table(x, y)
    )
)
